# IaC Lab 01 - Despliegue de Servidores Web

Este proyecto despliega dos servidores web independientes usando Python.

## Estructura
- `web01/`: Contiene el sitio para el puerto 4000.
- `web02/`: Contiene el sitio para el puerto 4001.

## Cómo ejecutar
Para levantar ambos servidores, ejecuta en la terminal:
bash deploy.sh
